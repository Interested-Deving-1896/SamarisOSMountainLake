# 23. Glossary

## 23.1 Terms and Definitions

| Term | Definition |
|------|------------|
| **Samaris OS** | Portable operating system based on Linux, Electron, React, and the VOLT native layer |
| **VOLT** | The native runtime spine and system orchestration layer of Samaris OS |
| **Kernel A** | The Node.js-based orchestrator (HTTP/WebSocket server on port 9999), message router, and service host |
| **Kernel B** | Alternative name for the Tesseract Engine — the Rust-based native acceleration daemon |
| **Tesseract Engine** | The Rust daemon providing low-level system acceleration, boot coordination, and IPC (package name: `tesseract-engine`) |
| **VRM** | Volt RAM Manager — deterministic memory manager with compression, deduplication, and pressure management |
| **VGM** | Volt GPU Manager — graphics adaptation layer with VRAM tiering, shader cache, and thermal backoff |
| **VUM** | Volt USB Manager — journaled storage manager with write caching and FUSE integration |
| **DWP** | Volt Dynamic Worker Pool — cooperative adaptive priority scheduler for background tasks |
| **ASC** | Adaptive System Configuration — hardware-aware policy compiler that generates system configuration at boot |
| **Volt Unifier** | Health monitoring, module registry, and coordination subsystem embedded in Kernel A |
| **SBP** | Samaris Binary Protocol — binary IPC protocol for typed, checksummed communication between VOLT components |
| **SBP-MEM** | SBP extension for memory management telemetry (pressure, quotas, compression state) |
| **AirBar** | The top system bar in the Samaris desktop UI (clock, status indicators, system menu) |
| **Peregrine** | The integrated native browser of Samaris OS, built on Electron WebContentsView |
| **Orbit** | The local AI assistant of Samaris OS, using Qwen3 LLM, Whisper STT, and OuteTTS |
| **BootSequence** | The VOLT boot initialisation procedure executed by the Tesseract Engine |
| **Readiness Signal** | A filesystem marker (`/run/volt-{service}.started`) or systemd notification indicating service availability |
| **Frame Guard** | DWP mechanism that protects desktop rendering by throttling background tasks when frame budget is exceeded |
| **Recovery Mode** | Minimal system mode activated after critical failures, providing diagnostic tools and repair options |
| **ASC Hardware Profile** | A named system configuration (balanced, performance, low_ram, etc.) generated by ASC based on detected hardware |
| **Pressure Zone** | A memory or GPU utilisation range (green, yellow, orange, red) that triggers proportional system responses |

## 23.3 Module Names vs. Systemd Units

| Module Name | systemd Service Unit | Binary Name |
|-------------|---------------------|-------------|
| Tesseract Engine (Kernel B) | `volt-kernel-b.service` | `tesseract-engine` |
| Kernel A | `volt-kernel.service` | `server.js` (Node.js) |
| VRM | `volt-ram-manager.service` | `volt-ram-manager` |
| VGM | `volt-gpu-manager.service` | `volt-gpu-manager` |
| VUM | `volt-usb-manager.service` | `volt-usb-manager` |
| DWP | `volt-worker-pool.service` | `volt-dynamic-worker-pool` |
| ASC | `volt-asc.service` | `volt-asc` |
| Volt Unifier (embedded) | `volt-unifier.service` | `volt-unifier-watchdog` (shell wrapper) |
| Desktop Shell | `volt-desktop.service` | `voltdesktop` (shell wrapper) |
| FS Service | `volt-fs.service` | `volt-fs-service` (shell wrapper) |

## 23.4 Key File System Paths

| Path | Content |
|------|---------|
| `/opt/volt/kernel/` | Kernel A orchestrator (Node.js) |
| `/opt/volt/kernel-b/` | Tesseract Engine configuration |
| `/opt/volt/electron/` | Electron desktop shell |
| `/opt/volt/desktop/` | React desktop SPA (built) |
| `/opt/volt/{asc,ram-manager,usb-manager,gpu-manager,worker-pool}/` | Module configurations |
| `/opt/volt/ai-models/` | AI model files (Qwen3, Whisper, OuteTTS) |
| `/run/samaris/` | Runtime state and generated configuration |
| `/var/lib/samaris/` | Persistent module data |
| `/var/log/samaris/` | Log files (conceptual — journald is primary) |
| `/etc/systemd/system/volt-*.service` | systemd unit files |
